# Alex Messias

**Segmento:** Concessionária automotiva — GWM (Great Wall Motors)  
**Status:** Projeto concluído (2026)  
**Produtos:** GWM H6, GWM H6 GT, ORA 5  
**Localização:** Brasil

---

## Perfil do negócio

Concessionária oficial GWM. Alto ticket — veículos de R$ 150k+. Público: classe média-alta interessada em SUVs e veículos elétricos/híbridos.

## O que foi entregue

**Campanha de posts** (5 criativos HTML → PNG exportados):

| Arquivo | Tema |
|---|---|
| campaign-01-hero.html | Hero institucional |
| campaign-02-produto.html | Destaque de produto |
| campaign-03-stat.html | Estatísticas / diferenciais |
| campaign-04-testdrive.html | CTA para test drive |
| campaign-05-institucional.html | Branding da concessionária |

**Assets visuais usados:**
- GWM H6 (azul, prata, branco, dark)
- GWM H6 GT (azul)
- ORA 5 (verde-teal, frente, interior, cinza, ação)
- Imagens de fábrica e lineup

**Stack:**
- Criativos em HTML/CSS (renderizados via Playwright → PNG)
- Scripts: `generate-posts.mjs` e `generate-posts-v2.mjs`

## Aprendizados

- HTML→PNG via Playwright funciona bem para criativos em escala
- Automação de geração de posts por scripts reutilizáveis é muito eficiente
- Para concessionárias: focar em test drive e diferenciais técnicos no criativo

- Landing page de test drive
