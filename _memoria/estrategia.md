# Estratégia — COO Framework

## Fase atual

**VENDA FIRST.** Nenhuma ferramenta, automação ou melhoria técnica é desenvolvida antes de ter receita entrando.

## Regra de decisão

Antes de qualquer tarefa, responder:
1. Isso fecha cliente nos próximos 30 dias?
2. Reduz trabalho operacional?
3. Aumenta lucro ou recorrência?

Se não: parar e voltar para vendas.

## Prioridade absoluta (essa semana)

Fechar o primeiro cliente do novo posicionamento. Qualquer valor acima de R$1.500/mês já valida a oferta.

## Hierarquia de trabalho

1. Vendas (prospecção, calls, propostas, follow-up)
2. Entrega de clientes ativos
3. Operação e processos
4. Tech e automações (só quando serve a 1, 2 ou 3)

## Tecnologia: quando desenvolver

- Cliente fechou → automatizar a entrega desse cliente
- Tarefa repetiu 3x → automatizar essa tarefa
- Nunca: "isso parece interessante" ou "vou preparar para o futuro"

## Nicho piloto

Clínicas de saúde (odonto, estética, médica) — Belém/PA primeiro, depois expandir.

## Modelo de crescimento

1. Fechar clientes com oferta única (Sistema Comercial Inteligente)
2. Padronizar entrega → template por cliente
3. Contratar primeiro operacional quando atingir R$30K
4. Escalar com o mesmo produto, não inventar novos
