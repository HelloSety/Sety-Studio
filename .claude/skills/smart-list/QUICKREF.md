# Quick Reference — Smart List (HubSpot)

## Comandos rapidos
- `Liste os deals [condições] e diga quantos são`
- `Segmente os contatos de [setor] com [condição]`
- `Exporte as empresas que [condição]`

## Ferramentas MCP
- `get_user_details` (preflight)
- `search_properties` / `get_properties` (achar a property certa + enums)
- `search_crm_objects` (filterGroups + `total`)
- `query_crm_data` (contagens / GROUP BY)

## Lembretes
- AND = mesmo filterGroup; OR = grupos separados.
- Sempre mostrar a property escolhida (nomes parecidos!) e o filtro usado.
- Somente leitura por padrão; escrita em lote vai para autofill/hygiene.
