# Saida e exportacao da lista

- Apresente a lista em tabela com os campos pedidos + link do registro (`urlTemplate`).
- SEMPRE informe o `total` (a busca pagina; o total reflete o conjunto inteiro).
- Mostre o filtro usado em texto, para o usuario auditar e reusar.
- Ofereca exportar (CSV/markdown) quando a lista for grande.
- Criar lista nativa (OBJECT_LIST) pode estar indisponivel para escrita no conector — nesse caso, entregue export + IDs e oriente criar no HubSpot.
- Para agir sobre a lista (tarefa em lote, atualizar property), encaminhe para `crm-autofill`/`crm-hygiene` com prévia + aprovação.
