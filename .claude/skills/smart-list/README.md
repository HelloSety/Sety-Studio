# smart-list (HubSpot)

Gera listas e segmentacoes a partir de linguagem natural — descobre property + operador certos, mostra a contagem (`total`) e entrega a lista. Resolve o caso de uso nº 1 de power users (montar filtros no HubSpot é confuso). Somente leitura por padrão; acoes em lote são delegadas às skills de escrita.
