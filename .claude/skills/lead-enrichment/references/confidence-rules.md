# Regras de confianca

- Alta (90-100%): site institucional / LinkedIn verificado.
- Media (60-89%): portais de noticias/vagas conhecidos (Gupy, Indeed), diretorios oficiais.
- Baixa (<60%): blogs de terceiros, foruns, estimativas geradas por IA.

Dados de baixa confianca NAO vao para properties permanentes — apenas para `NOTE`. Sempre exiba a fonte e a confianca por campo.
