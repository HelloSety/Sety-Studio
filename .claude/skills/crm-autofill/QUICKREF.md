# Quick Reference — CRM Autofill (HubSpot)

## Comandos rapidos
- `Registrar reuniao no deal [Nome]`
- `Criar proxima tarefa para amanha`
- `Extrair dados da transcricao`

## Ferramentas MCP
- `search_crm_objects` (identificar deal / antiduplicidade, query e associatedWith)
- `search_properties` (properties existentes)
- `manage_crm_objects` (NOTE, TASK, update do deal; lote <=10)

## Tarefas (HubSpot)
- Aberta = `hs_task_status: NOT_STARTED`
- Tipos: CALL, EMAIL, MEETING, TODO, LINKED_IN...
- Vencimento: `hs_timestamp`
