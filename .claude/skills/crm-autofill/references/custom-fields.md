# Campos personalizados (HubSpot)

- A skill NAO cria properties novas. So preenche as que ja existem.
- Descubra via `search_properties` (nomes internos) e `get_properties` (tipos/enums).
- Se a informacao nao tiver property correspondente, sugira: criar a property no HubSpot, mapear para equivalente, salvar como `NOTE`, ou ignorar.
- Enums exigem o `value` interno exato.
