# crm-hygiene (HubSpot)

Audita a higiene dos REGISTROS do HubSpot e calcula um CRM Health Score, com plano de correcao executavel sob aprovacao. Para higiene de ESQUEMA (properties), veja a skill `data-model-audit`.

Referencias em `references/`: object model, tool map, operation scope, preflight, write safety, HubSQL cheatsheet, health score, taxonomia de problemas e regras de correcao em lote.
