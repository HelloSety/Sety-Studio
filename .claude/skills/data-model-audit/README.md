# data-model-audit (HubSpot) — flagship

Higiene de ESQUEMA (properties), nao de registros: property sprawl, duplicadas, sem uso, mal nomeadas, read-only. Calcula um Schema Health Score e um plano de consolidacao. Diferencial de topo de gama do HubSpot (o nativo so cobre no Ops Hub/Enterprise). Quando o MCP nao permitir alterar o schema, entrega o plano para execucao no HubSpot Settings.
