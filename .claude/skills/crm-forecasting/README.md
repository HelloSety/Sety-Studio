# crm-forecasting (HubSpot)

Forecast de cenarios (Expected/Best/Worst) com a probabilidade NATIVA do HubSpot (`hs_deal_stage_probability`, `hs_projected_amount`), gap contra meta e plano de aceleracao. Diferente de RD/Pipedrive, nao usa tabela de probabilidade hardcoded — usa a nativa (com override opcional).
